The Postgram name, logos, wordmarks, icons, and other distinctive brand assets

this repository do not grant permission to use those marks except as described
here or as required by applicable law.

## Uses that do not require permission

You may:

- truthfully state that software works with, connects to, or is based on
  Postgram;
- link to the official project and use the Postgram name in ordinary text to
  identify it;
- use the unmodified Postgram name and included branding when distributing an
  unmodified official release, provided you do not imply extra endorsement;
- use the marks in reviews, commentary, education, or other nominative and fair
  uses permitted by law.

## Uses that require permission

Without prior written permission, you may not:

- use Postgram or a confusingly similar name as the name of a modified fork,
  product, service, company, domain, package, or application;
- use Postgram logos or icons to brand a modified distribution or hosted
  service;
- imply that a fork, integration, service, event, or organization is official,
  sponsored, certified, or endorsed by the Postgram project;
- alter the marks in a way that is misleading or harmful to the project.

A modified distribution may describe itself as "based on Postgram" if that
description is accurate, secondary to its own name, and does not imply official
status. Modified user interfaces should remove Postgram logos and clearly state
that the software has been modified.

## Permission requests

Request permission by contacting the repository owner through the contact
methods on their GitHub profile. Include the proposed name, visual treatment,
context, and expected duration of use.

This policy concerns trademark and brand use only. It does not change or add a
restriction to the AGPL, MIT, or CC BY licensed materials in this repository.
