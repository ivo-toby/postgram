# Phase 1 MVP Implementation Plan

> **For agentic workers:** REQUIRED: Use
> `superpowers:subagent-driven-development` (if subagents available) or
> `superpowers:executing-plans` to implement this plan. Steps use checkbox
> (`- [ ]`) syntax for tracking.

**Goal:** Build the Phase 1 central knowledge store from the approved spec
package with async enrichment, REST + MCP transports, CLI/admin tooling, and
contract-verified behavior.

**Architecture:** Implement a service-layer-first TypeScript Node service around
Postgres + pgvector. Keep Hono REST handlers and MCP tools thin, make the human
CLI call REST, and keep admin operations container-local with direct DB access.

**Tech Stack:** TypeScript, Node 22, Hono, pg, pgvector, zod, pino, neverthrow,
Vitest, commander, argon2, OpenAI SDK
