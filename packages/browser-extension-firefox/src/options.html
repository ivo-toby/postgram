Postgram Web Clipper

Capture the current page (or a text selection) and save it to your self-hosted Postgram server.

Server
Capture defaults

How it works

  1. Configure the endpoint and API key above, then click Save.
  2. Click Grant access so the extension can reach your server (the browser will prompt you).
  3. Visit any page and click the Postgram toolbar icon. If you have text selected, only the selection is captured; otherwise the whole page is captured.
